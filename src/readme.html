Installing the project

  • - 1. Clone the repository git clone https://raphaelcrv@bitbucket.org/raphaelcrv/le-musicas.git
  • - 2. change to the le-musicas root folder and install the bower dependencies bower install
    after done install node dependencies npm install

Running the project

  • run the project with the command npm run server after loaded the project will be avaiable

    Local: http://localhost:3000
    External: http://yourIPaddress:3000
    -----------------------------------
    UI: http://localhost:3001
    UI External: http://yourIPaddress:3001

Running the project

  • for generate the distribuition version use the commnad gulp build

    Use NPM scripts

    • npm run build to build an optimized version of your application in /dist
    • npm run serve to launch a browser sync server on your source files
    • npm run serve:dist to launch a server on your optimized application
    • npm run test to launch your unit tests with Karma
    • npm run test:auto to launch your unit tests with Karma in watch mode
